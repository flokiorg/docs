## Installation

1.  **Download** the package for your operating system using the buttons above.
2.  **Execute the installer**:
    *   **Windows**: Run the `.exe` and follow the setup wizard.
    *   **macOS**: Open the `.dmg` and drag Lokinode to your Applications folder.
    *   **Linux**: Install the `.deb` package via your package manager.
3.  **Setup**: Launch the app to initialize your node. You can create a new 24-word seed phrase or import an existing wallet.

## Capabilities

### Lightning Network (FLN) Operations
Lokinode simplifies advanced Lightning Network management:
*   **Visual Channel Control**: Open, close, and monitor channel states without terminal commands.
*   **Liquidity Management**: Track inbound and outbound capacity with real-time visual feedback.
*   **Settlement**: Generate and pay Lightning invoices with instant network confirmation.

### Persistent Experience
Designed for node reliability, Lokinode separates the node process from the graphic interface:
*   **System Tray Integration**: Control node state and view quick status updates from the OS tray.
*   **Non-Intrusive Runtime**: Closing the application window hides the UI but keeps the node active in the background.
*   **Auto-Stability**: Built-in logic to handle network interruptions and maintain peer connectivity.

### Flexible Configuration
Lokinode is built for power users who require control over their environment:
*   **Custom Paths**: Specify any local or external directory for blockchain and channel data storage.
*   **Real-time Metrics**: Monitor block height, sync progress, and peer count directly from the dashboard.
*   **Security Suite**: Manage encrypted wallet storage and recovery credentials within a secure local environment.

## Managing with Lokihub

While **Lokinode** provides a built-in interface for basic operations, advanced operators can use **Lokihub** for detailed routing analytics and remote node management.

### Quick Connection
1.  **Active Node**: Ensure Lokinode is running and synchronized.
2.  **Connect**: Open Lokihub; it will automatically attempt to detect credentials from your active Lokinode directory.
3.  **Synchronize**: Gain access to advanced liquidity graphs and routing history.

[View the Lokihub Setup Guide →](/lokihub/setup)

## Data Locations

Lokinode uses standard paths by default, but allows for full customization during the initial setup or via settings.

| OS | Default Path |
| :--- | :--- |
| **Linux** | `~/.flnd/` |
| **macOS** | `~/Library/Application Support/Flnd/` |
| **Windows** | `%APPDATA%\Flnd\` |

## CRITICAL: Lightning Backups

Your 24-word seed phrase **only recovers on-chain funds**. 

1.  **Static Channel Backups (SCB)**: A `channel.backup` file is required to recover funds from open Lightning channels.
2.  **Safety Requirement**: You must maintain a separate backup of the `channel.backup` file alongside your seed phrase.

:::danger[DATA LOSS RISK]
Always ensure you have a verified backup of your `channel.backup` file. Without it, Lightning channel funds cannot be recovered if local data is lost.
:::
