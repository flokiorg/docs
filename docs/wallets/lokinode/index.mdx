## Installation & Setup

1.  **Download** the installer or package for your operating system.
2.  **Run the application**:
    *   **Windows**: Run the `.exe` installer.
    *   **macOS**: Open the `.dmg` and drag Lokinode to your Applications folder.
    *   **Linux**: Install the `.deb` package using `sudo dpkg -i lokinode_*.deb`.
3.  **Initialize Wallet**: On first launch, you will be prompted to create a new wallet or restore an existing one using a 24-word seed phrase.
4.  **Sync Node**: Lokinode will start the internal **FLND** node. Wait for it to catch up with the Flokicoin network.

## Key Capabilities

### <Activity size={24} style={{verticalAlign: 'middle', marginRight: '8px'}} /> Professional Dashboard
Get a high-level overview of your node's health. Monitor synchronization progress, peer connections, and network stats in real-time without touching the command line.

### <Zap size={24} style={{verticalAlign: 'middle', marginRight: '8px'}} /> Wallet & Transactions
*   **Send & Receive**: Intuitive forms for FLC transactions with built-in fee estimation.
*   **History**: Searchable and filterable transaction logs for both on-chain and lightning payments.
*   **Security**: Manage your 24-word seed phrase and wallet passwords securely.

### <Settings size={24} style={{verticalAlign: 'middle', marginRight: '8px'}} /> Advanced Management
Access deep node configurations through a structured settings menu. View internal logs, manage RPC credentials, and tune your node's performance to match your hardware.
