# tWallet v0 Release History

This page tracks the history of **tWallet v0 (Electrum)**. The v0 branch is maintained for users who require **BIP39 seed restoration** or prefer using public Flokicoin Electrum servers.

| Version | Focus |
| :--- | :--- |
| [**0.1.1-alpha**](/wallets/twallet/v0/0.1.1-alpha) | Added multi-server failover and improved wallet synchronization logic. |
| [**0.1.0-alpha**](/wallets/twallet/v0/0.1.0-alpha) | First public alpha for the terminal interface with Electrum support. |

---

:::important[Looking for v1?]
If you don't need to restore an existing BIP39 seed, we recommend checking out [**tWallet v1**](/wallets/twallet/v1) for the best performance and privacy (Neutrino).
:::
