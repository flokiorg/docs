## Quick Comparison

| Feature | tWallet | Komodo Wallet |
| :--- | :--- | :--- |
| **Interface** | Terminal (TUI) | Graphic (GUI) |
| **Lightning** | Yes (Built-in) | No |
| **Experience** | Intermediate | Beginner |
| **Best For** | Node operators & Privacy | Everyday users & Mobile |

:::tip[Running Lokihub?]
If you plan on using **Lokihub**, we highly recommend starting with **tWallet v1**, as it provides the necessary Lightning backend for node operations.
:::
