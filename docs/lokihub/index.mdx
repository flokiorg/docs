**Lokihub** is a self-custodial interface for managing your Lightning node. Connect to the FLND (Flokicoin Lightning Network Daemon) node integrated within [tWallet](/wallets/twallet), create sub-wallets, handle channels, connect apps via [NWC](https://nwc.dev), and manage your peers.
