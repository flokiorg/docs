## Instalación y Uso

:::tip
Si su sistema operativo marca el binario como no verificado, es posible que deba permitirlo explícitamente en sus ajustes de seguridad.
:::

### Escritorio (GUI)

**macOS**
1. Abra el archivo `.dmg` descargado.
2. Arrastre **Lokihub** a su carpeta de Aplicaciones.

**Windows**
1. Extraiga el archivo `.zip` descargado.
2. Ejecute `lokihub.exe`.

**Linux**
- **AppImage**:
  ```bash
  chmod +x lokihub-*.AppImage
  ./lokihub-*.AppImage
  ```
- **Debian/Ubuntu**:
  ```bash
  sudo dpkg -i lokihub-*.deb
  ```
- **Fedora/RHEL**:
  ```bash
  sudo rpm -i lokihub-*.rpm
  ```

### Servidor (HTTP)

La versión de servidor se ejecuta como un proceso de fondo y sirve la interfaz web a través de HTTP.

1. **Extraer el archivo**:
   ```bash
   tar -xvf lokihub-server-*.tar.gz  # Linux
   # O descomprimir en Windows/macOS
   ```

2. **Ejecutar el binario**:
   ```bash
   ./lokihub
   ```

3. **Abrir en el Navegador**:
   Navegue a `http://localhost:1610` (puerto por defecto).
