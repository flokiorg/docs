## Comparación Rápida

| Característica  | tWallet           | Komodo Wallet       |
| :---            | :---              | :---                |
| **Interfaz**    | Terminal (TUI)    | Gráfica (GUI)       |
| **Lightning**   | Sí (Integrada)    | No                  |
| **Experiencia** | Intermedio        | Principiante        |
| **Recomendado para** | Operadores de nodos y privacidad | Usuarios cotidianos y móvil |

:::tip[¿Usas Lokihub?]
Si planeas usar **Lokihub**, te recomendamos ampliamente comenzar con **tWallet v1**, ya que proporciona el backend Lightning necesario para las operaciones del nodo.
:::
