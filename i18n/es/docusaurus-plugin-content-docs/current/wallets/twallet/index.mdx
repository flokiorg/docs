## Primeros pasos

Una vez que hayas elegido una versión, la configuración toma unos 5 minutos:

1. **Descarga** la versión para tu sistema operativo.
2. **Respalda** tu frase de recuperación. Anótala y **nunca la compartas**.
3. **Sincroniza**: deja que la cartera se ponga al día con la red (unos pocos minutos como máximo).
4. **Recibe**: genera una dirección y obtén tus primeras monedas.
