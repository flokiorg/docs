Este lanzamiento se enfoca en actualizar las dependencias centrales para integrar completamente `flnd v0.1.21-beta`, habilitando el soporte para canales Taproot listos para producción y resolviendo la compatibilidad con plataformas de 32 bits.

## Cambios notables

### Sincronización del núcleo
Actualizado a la última versión de nuestro demonio lightning principal:
- **flnd**: Actualizado a `v0.1.21-beta` (canales Taproot y correcciones de desbordamiento de 32 bits).

### Mejoras
- Limpiezas internas rutinarias y optimizaciones de rendimiento.
- Se actualizaron las dependencias del módulo y se organizó `go.mod`.
