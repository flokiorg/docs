# Historial de lanzamientos de tWallet v1

Esta página rastrea la evolución de **tWallet v1**. Aunque recomendamos usar siempre la última versión por seguridad y rendimiento, puede encontrar el historial completo de las versiones pre-lanzamiento de la v1 a continuación.

| Versión | Fecha | Cambios clave |
| :--- | :--- | :--- |
| [**1.0.9-beta**](/wallets/twallet/v1/1.0.9-beta) | Última | Se agregaron activos de Windows de 32 bits y se mejoró la gestión de dependencias. |
| [**1.0.8-beta**](/wallets/twallet/v1/1.0.8-beta) | - | Migración de Neutrino de btcsuite al fork de Flokiorg para una mayor confiabilidad. |
| [**1.0.5-beta**](/wallets/twallet/v1/1.0.5-beta) | - | Corrección crítica para el escaneo de transacciones en dispositivos con poca memoria. |
| [**1.0.4-beta**](/wallets/twallet/v1/1.0.4-beta) | - | Flujo gRPC optimizado para clientes de panel de control compatibles con dispositivos móviles. |
| [**1.0.3-alpha**](/wallets/twallet/v1/1.0.3-alpha) | - | Implementación inicial del protocolo Flokicoin Neutrino. |
| [**1.0.2-alpha**](/wallets/twallet/v1/1.0.2-alpha) | - | Se agregó soporte para direcciones Segwit y Taproot. |
| [**1.0.1-alpha**](/wallets/twallet/v1/1.0.1-alpha) | - | Primera prueba pública del servicio WalletKit gRPC. |

---

:::tip[Manténgase siempre actualizado]
Para obtener la mejor experiencia y los últimos parches de seguridad, recomendamos encarecidamente ejecutar la última versión desde nuestros [Lanzamientos en GitHub](https://github.com/flokiorg/tWallet/releases).
:::
