Esta es una **versión pre-lanzamiento** para pruebas y retroalimentación. Se anima a los desarrolladores y primeros adoptantes a **informar problemas**.

## Cambios

- Se registró el servicio gRPC de WalletKit (`walletrpc.WalletKit`), por lo que los puntos de enlace de WalletKit ya están disponibles.
