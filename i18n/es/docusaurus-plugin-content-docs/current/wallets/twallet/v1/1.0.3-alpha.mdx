Esta es una **versión pre-lanzamiento** para pruebas y retroalimentación. Se anima a los desarrolladores y primeros adoptantes a **informar problemas**.

## Registro de cambios

- Migración del backend de **Electrum** a **Neutrino**
- Integración del **Nodo Lightning de Flokicoin (LN)** con soporte completo de la pila LN
- Permite el **desarrollo nativo de Lightning** en la red Flokicoin
- Se agregaron tipos de dirección configurables: `segwit`, `nested-segwit`, `taproot`
- Introducción de una interfaz de usuario segura para el bloqueo/desbloqueo de carteras y el cambio de contraseña
- Mejora del constructor de transacciones con estimación de comisiones en vivo
- Mejora de la **visualización del saldo** con montos confirmados y no confirmados
- Interfaz de usuario de terminal (TUI) mejorada con estado de sincronización, salud de LN y actualizaciones en vivo
- Refactorización interna mayor para configuración modular, estabilidad y expansión futura
