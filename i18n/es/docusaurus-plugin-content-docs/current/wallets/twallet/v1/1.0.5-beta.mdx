## Cambios

### Dependencias
- Actualizado `flokicoin-neutrino` → **v0.16.3-beta**

### Correcciones
- Depende de neutrino con puntos de control de filtros basados en parámetros (params‑sourced filter checkpoints).

## Notas
- Este lanzamiento se alinea con las últimas actualizaciones de las librerías originales.
- Se recomienda a los usuarios actualizar para obtener una mejor estabilidad y compatibilidad.
