- Esta es una **versión pre-lanzamiento** para pruebas y retroalimentación.
- Se anima a los desarrolladores y primeros adoptantes a **informar problemas**.
- Opciones de configuración de comisiones: `feeslow`, `feemedium`, `feefast`
- Se cambió el nombre del directorio de datos de la aplicación de `flcwallet` a `twallet`
