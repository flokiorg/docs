# Historial de lanzamientos de tWallet v0

Esta página rastrea el historial de **tWallet v0 (Electrum)**. La rama v0 se mantiene para usuarios que requieren la **restauración de semillas BIP39** o prefieren usar servidores públicos de Electrum de Flokicoin.

| Versión | Enfoque |
| :--- | :--- |
| [**0.1.1-alpha**](/wallets/twallet/v0/0.1.1-alpha) | Se añadió la conmutación por error multiservidor y se mejoró la lógica de sincronización de la cartera. |
| [**0.1.0-alpha**](/wallets/twallet/v0/0.1.0-alpha) | Primera alfa pública para la interfaz de terminal con soporte para Electrum. |

---

:::important[¿Busca la v1?]
Si no necesita restaurar una semilla BIP39 existente, le recomendamos consultar [**tWallet v1**](/wallets/twallet/v1) para obtener el mejor rendimiento y privacidad (Neutrino).
:::
