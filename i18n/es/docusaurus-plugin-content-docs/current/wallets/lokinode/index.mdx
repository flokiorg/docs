Lokinode es una aplicación de escritorio multiplataforma que proporciona una interfaz gráfica para ejecutar y administrar un nodo completo de Flokicoin (FLND). Te permite operar una billetera on-chain y participar en la Red Lightning de Flokicoin (FLN).

## Características

- **Servicio en segundo plano**: Lokinode se ejecuta como un servicio persistente. Puedes cerrar la interfaz de usuario mientras el nodo continúa ejecutándose en la bandeja del sistema.
- **Billetera on-chain**: Inicializa y desbloquea tu billetera, envía/recibe FLC y consulta el historial de transacciones.
- **Gestión de nodos**: Supervisa la altura del bloque, el estado de sincronización y el tip del mempool. Visualiza los registros de la aplicación directamente en la interfaz.
- **Perfiles multi-nodo**: Ejecuta y gestiona varios nodos en la misma máquina. Lokinode rastrea automáticamente tus directorios de datos y configuraciones, lo que te permite cambiar rápidamente entre diferentes perfiles de nodo.

## Instalación y configuración

:::tip[¿Vienes de tWallet?]
Si ya usas [**tWallet**](/wallets/twallet), **no es necesaria ninguna migración**. Tu frase semilla de 24 palabras es totalmente compatible. Mejor aún, debido a que ambas aplicaciones comparten los mismos directorios de datos predeterminados, Lokinode detectará automáticamente tu nodo existente. Simplemente puedes iniciar la aplicación y encender tu perfil de nodo predeterminado con **cero configuración requerida**.
:::

1. **Descarga**: Descarga el ejecutable para tu sistema operativo correspondiente.
2. **Configuración del nodo**: Inicia Lokinode. El asistente de configuración te pedirá primero que establezcas un alias público y selecciones un directorio de datos local.
3. **Configuración de la billetera**: Elige entre crear una nueva billetera (que proporciona una nueva frase semilla de 24 palabras) o restaurar una existente mediante un mnemónico.
4. **Seguridad**: Establece una contraseña de cifrado sólida para proteger tus datos locales.
5. **Sincronización**: Una vez que el nodo arranque, comenzará automáticamente a sincronizarse con la cadena de bloques de Flokicoin.

## Gestión con Lokihub

Mientras que Lokinode se encarga de ejecutar tu nodo Lightning en segundo plano y proporciona una interfaz para operaciones básicas de billetera on-chain, debes usar **Lokihub** para las funciones avanzadas de la Red Lightning.

Cuando abras Lokihub en la misma máquina, intentará conectarse automáticamente a tu directorio activo de Lokinode, desbloqueando las siguientes capacidades:

- **Gestión de liquidez**: Abre, cierra y supervisa los canales Lightning para gestionar tu capacidad entrante y saliente.
- **Sub-billeteras**: Crea billeteras dedicadas para organizar tus fondos para diferentes aplicaciones o presupuestos personales.
- **Integraciones de aplicaciones (NWC)**: Conecta aplicaciones externas a tu nodo de forma fluida mediante Nostr Wallet Connect, con controles de permisos granulares.

[Ver la Guía de configuración de Lokihub →](/lokihub/setup)

## Ubicaciones de datos

Por defecto, Lokinode almacena los datos de la cadena de bloques y de la billetera en los siguientes directorios:

| SO | Ruta predeterminada |
| :--- | :--- |
| **Linux** | `~/.flnd/` |
| **macOS** | `~/Library/Application Support/Flnd/` |
| **Windows** | `%APPDATA%\Flnd\` |

Más allá de estas rutas predeterminadas, puedes crear un nuevo perfil en una ubicación personalizada o mover un perfil existente copiando su directorio de datos a una nueva ruta y volviéndolo a abrir a través de la gestión de perfiles de la aplicación.

## CRÍTICO: Respaldos de Lightning

Tu frase semilla de 24 palabras **solo recupera fondos on-chain**.

Para recuperar fondos bloqueados en canales Lightning abiertos, necesitas un **Respaldo Estático de Canales (SCB)**.

:::danger[RIESGO DE PÉRDIDA DE DATOS]
Asegúrate siempre de tener un respaldo separado y verificado de tu archivo `channel.backup` junto con tu frase semilla. Sin él, los fondos de los canales Lightning no se pueden recuperar si se pierden los datos locales.
:::
