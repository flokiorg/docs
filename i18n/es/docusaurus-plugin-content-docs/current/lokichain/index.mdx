Lokichain es la cadena de bloques detrás de Flokicoin. Valida cada transacción, aplica las reglas de consenso y mantiene todo descentralizado. Si quieres entender el motor bajo el capó, estás en el lugar correcto.

:::release
**Última versión**: Flokicoin Core [v0.25.13-alpha](/lokichain/releases/0.25.13-alpha) — nuestra implementación de referencia actual.
:::

## Ejecutar un Nodo
Ejecutar un nodo completo es la forma más directa de apoyar la red. Tu nodo verifica toda la cadena de forma independiente y sirve RPCs para monederos, mineros y aplicaciones.

:::tip[Opción rápida]
¿Buscas una configuración más sencilla? Prueba el nodo plug-and-play comunitario en [myfloki.com/fullnode](https://myfloki.com/fullnode).
:::

## Minar
Los mineros aportan el poder de hash que hace que la cadena sea inmutable. Necesitarás el software de nodo más reciente y un ASIC Scrypt o GPU.

- **Comenzar** — [Guía de configuración de minería](/mining).

## Cómo Funciona
Lokichain es una red peer-to-peer. Cada nodo aplica las mismas reglas, por lo que no existe un único punto de fallo. Las reglas de consenso se actualizan mediante FIPs — considéralos como las "leyes" de la red.
