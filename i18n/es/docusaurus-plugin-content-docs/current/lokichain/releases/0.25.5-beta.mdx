## Cambios notables

- Se agregaron nuevos puntos de control (checkpoints) de la cadena de bloques, incluyendo el bloque 0 y alturas posteriores.
- Manejo mejorado de PSBT: soporte para llaves públicas extendidas (XPubs) y validación de rutas.
- Se agregó cobertura de pruebas PSBT para la serialización/deserialización de rutas BIP32.
- Validación mejorada de la longitud mínima de la ruta PSBT.
- Validación mejorada de la marca de tiempo de la cabecera del bloque.
- Se actualizaron mempool, txscript y wallet para usar `maps.Copy` para un manejo de mapas más seguro y limpio.
- Error tipográfico corregido: se corrigió la constante PSBT de `XpubType` a `XPubType`.
- Se corrigieron problemas menores en la codificación/decodificación de PSBT y en los comentarios.
- Se renombró `flokicoin-cli` a `flokicoind-cli` en todos los binarios, rutas de configuración y documentación.
