## Installation & Utilisation

:::tip
Si votre système d'exploitation signale le binaire comme non vérifié, vous devrez peut-être l'autoriser explicitement dans vos paramètres de sécurité.
:::

### Bureau (GUI)

**macOS**
1. Ouvrez le fichier `.dmg` téléchargé.
2. Faites glisser **Lokihub** vers votre dossier Applications.

**Windows**
1. Extrayez le fichier `.zip` téléchargé.
2. Lancez `lokihub.exe`.

**Linux**
- **AppImage** :
  ```bash
  chmod +x lokihub-*.AppImage
  ./lokihub-*.AppImage
  ```
- **Debian/Ubuntu** :
  ```bash
  sudo dpkg -i lokihub-*.deb
  ```
- **Fedora/RHEL** :
  ```bash
  sudo rpm -i lokihub-*.rpm
  ```

### Serveur (HTTP)

La version serveur s'exécute comme un processus en arrière-plan et sert l'interface web via HTTP.

1. **Extraire l'archive** :
   ```bash
   tar -xvf lokihub-server-*.tar.gz  # Linux
   # Ou dézippez sur Windows/macOS
   ```

2. **Lancer le binaire** :
   ```bash
   ./lokihub
   ```

3. **Ouvrir dans le navigateur** :
   Naviguez vers `http://localhost:1610` (port par défaut).
