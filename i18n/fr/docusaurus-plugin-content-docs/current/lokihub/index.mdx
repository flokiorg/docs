# Maîtrisez Votre Nœud Lightning

**Lokihub** vous confère une souveraineté financière totale sur votre nœud Lightning. Au-delà d'un simple gestionnaire, c'est votre passerelle vers le web décentralisé — intégré à [tWallet](/wallets/twallet) pour vous aider à gérer des sous-portefeuilles, administrer des canaux et connecter des applications via NWC.

:::caution[CRITIQUE : RISQUE DE SAUVEGARDE]
**Les fonds des canaux Lightning NE sont PAS stockés sur la blockchain.** Une phrase mnémotechnique de 24 mots ne récupère que votre solde on-chain. Pour éviter la perte totale de vos fonds de canal, vous devez sauvegarder vos **Static Channel Backups (SCB)**.
[Apprenez à protéger votre nœud →](/wallets/twallet/v1#critical-lightning-backups)
:::
