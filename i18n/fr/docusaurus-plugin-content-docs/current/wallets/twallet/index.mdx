## Pour commencer

Une fois que vous avez choisi une version, la configuration prend environ 5 minutes :

1. **Téléchargez** la version correspondant à votre système d'exploitation.
2. **Sauvegardez** votre phrase de récupération. Écrivez-la et **ne la partagez jamais**.
3. **Synchronisez** — laissez le portefeuille rattraper le réseau (quelques minutes au maximum).
4. **Recevez** — générez une adresse et recevez vos premières pièces.
