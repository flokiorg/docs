- Ceci est une **version préliminaire** pour tests et commentaires.
- Les développeurs et les premiers adoptants sont encouragés à **signaler les problèmes**.
- Options de configuration des frais : `feeslow`, `feemedium`, `feefast`.
- Répertoire de données d'application renommé de `flcwallet` à `twallet`.
