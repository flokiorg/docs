# Historique des Versions de tWallet v0

Cette page suit l'historique de **tWallet v0 (Electrum)**. La branche v0 est maintenue pour les utilisateurs qui nécessitent une **restauration de graine BIP39** ou qui préfèrent utiliser des serveurs Electrum Flokicoin publics.

| Version | Focus |
| :--- | :--- |
| [**0.1.1-alpha**](/wallets/twallet/v0/0.1.1-alpha) | Ajout du basculement multi-serveur et amélioration de la logique de synchronisation du portefeuille. |
| [**0.1.0-alpha**](/wallets/twallet/v0/0.1.0-alpha) | Première version alpha publique pour l'interface du terminal avec support Electrum. |

---

:::important[Vous cherchez la v1 ?]
Si vous n'avez pas besoin de restaurer une graine BIP39 existante, nous vous recommandons de consulter [**tWallet v1**](/wallets/twallet/v1) pour obtenir les meilleures performances et une confidentialité accrue (Neutrino).
:::
