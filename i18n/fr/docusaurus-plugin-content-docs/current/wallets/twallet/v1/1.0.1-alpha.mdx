Ceci est une **version préliminaire** pour tests et commentaires. Les développeurs et les premiers adoptants sont encouragés à **signaler les problèmes**.

## Changements

- Enregistrement du service WalletKit gRPC (`walletrpc.WalletKit`) afin que les points de terminaison WalletKit soient désormais disponibles.
