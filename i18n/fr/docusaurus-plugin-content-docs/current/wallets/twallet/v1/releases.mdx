# Historique des Versions de tWallet v1

Cette page suit l'évolution de **tWallet v1**. Bien que nous recommandions de toujours utiliser la dernière version pour des raisons de sécurité et de performance, vous trouverez ci-dessous l'historique complet des versions préliminaires de v1.

| Version | Date | Changements Clés |
| :--- | :--- | :--- |
| [**1.0.9-beta**](/wallets/twallet/v1/1.0.9-beta) | Dernier | Ajout d'actifs Windows 32 bits et amélioration de la gestion des dépendances. |
| [**1.0.8-beta**](/wallets/twallet/v1/1.0.8-beta) | - | Migration de Neutrino de btcsuite vers le fork de Flokiorg pour une fiabilité accrue. |
| [**1.0.5-beta**](/wallets/twallet/v1/1.0.5-beta) | - | Correction critique pour le scan des transactions sur les appareils à faible mémoire. |
| [**1.0.4-beta**](/wallets/twallet/v1/1.0.4-beta) | - | Flux gRPC optimisé pour les clients de tableau de bord compatibles mobiles. |
| [**1.0.3-alpha**](/wallets/twallet/v1/1.0.3-alpha) | - | Mise en œuvre initiale du protocole Flokicoin Neutrino. |
| [**1.0.2-alpha**](/wallets/twallet/v1/1.0.2-alpha) | - | Ajout du support des adresses Segwit et Taproot. |
| [**1.0.1-alpha**](/wallets/twallet/v1/1.0.1-alpha) | - | Premier test public du service WalletKit gRPC. |

---

:::tip[Restez Toujours à Jour]
Pour une expérience optimale et les derniers correctifs de sécurité, nous vous recommandons vivement d'utiliser la dernière version disponible sur nos [Releases GitHub](https://github.com/flokiorg/tWallet/releases).
:::
