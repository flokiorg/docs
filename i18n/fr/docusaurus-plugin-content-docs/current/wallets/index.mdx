## Comparaison Rapide
| Fonctionnalité | tWallet | Komodo Wallet |
| :--- | :--- | :--- |
| **Interface** | Terminal (TUI) | Graphique (GUI) |
| **Lightning** | Oui (Intégré) | Non |
| **Expérience** | Intermédiaire | Débutant |
| **Idéal Pour** | Opérateurs de nœuds & Confidentialité | Utilisateurs du quotidien & Mobile |

:::tip[Vous utilisez Lokihub ?]
Si vous prévoyez d'utiliser **Lokihub**, nous recommandons vivement de commencer avec **tWallet v1**, car il fournit le backend Lightning nécessaire aux opérations de nœud.
:::
