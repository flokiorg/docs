Flokicoin Core est le nœud complet de référence pour Lokichain. Il valide les blocs et les transactions, applique le consensus et expose des RPC pour les portefeuilles et les services.

Veuillez signaler les bogues sur GitHub :
https://github.com/flokiorg/go-flokicoin/issues
