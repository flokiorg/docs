Lokichain est la blockchain qui sous-tend Flokicoin. Elle valide chaque transaction, applique les règles de consensus et maintient la décentralisation. Si vous souhaitez comprendre le moteur sous le capot, vous êtes au bon endroit.

:::release
**Dernière version** : Flokicoin Core [v0.25.12-alpha](/lokichain/releases/0.25.12-alpha) — notre implémentation de référence actuelle.
:::

## Lancer un Nœud
Faire tourner un nœud complet est la façon la plus directe de soutenir le réseau. Votre nœud vérifie l'intégralité de la chaîne de manière indépendante et expose des RPC pour les portefeuilles, les mineurs et les applications.

:::tip[Option simplifiée]
Vous préférez une configuration plus simple ? Essayez le nœud plug-and-play communautaire sur [myfloki.com/fullnode](https://myfloki.com/fullnode).
:::

## Développer sur Lokichain
Que vous construisiez un portefeuille, un explorateur ou quelque chose de nouveau — voici par où commencer :
- **FIPs** — Suivez l'évolution du protocole via les [Flokicoin Improvement Proposals](/lokichain/fips).

- **Code source** — Parcourez et contribuez à [go-flokicoin](https://github.com/flokiorg/go-flokicoin).

## Miner
Les mineurs fournissent la puissance de hachage qui rend la chaîne immuable. Vous aurez besoin du dernier logiciel de nœud et d'un ASIC Scrypt ou d'un GPU.
- **Démarrer** — [Guide de configuration du minage](/mining).

## Comment Ça Fonctionne
Lokichain est un réseau pair-à-pair. Chaque nœud applique les mêmes règles, il n'y a donc aucun point de défaillance unique. Les règles de consensus sont mises à jour via les FIPs — considérez-les comme les « lois » du réseau.
