## Pools
- [fpool.net](https://fpool.net) <span style={{display: 'inline-flex', alignItems: 'center', gap: '0.25rem', verticalAlign: 'middle'}}><Fish size={16} /> Le minage n'a jamais été aussi facile</span> (solo, PPLNS)
- [viaflc.com](https://viaflc.com) Plus de pièces pour la même puissance (PPLNS, merge-mining)


### Notes
- Mainnet : utilisez un GPU ou un ASIC scrypt. Le minage CPU n'est pas compétitif ; réservez cpuminer pour le testnet ou l'expérimentation.
- AuxPoW : si le pool prend en charge le merge mining avec une chaîne parente de la famille scrypt et transmet l'AuxPoW à Flokicoin, suivez les instructions spécifiques du pool. Les détails varient selon le pool.


## Bases de Stratum
- URL : `stratum+tcp://HÔTE_POOL:PORT`
- Nom d'utilisateur : votre adresse de paiement Flokicoin (ou compte de pool)
- Mot de passe : souvent `x` ou un jeton spécifique au pool
