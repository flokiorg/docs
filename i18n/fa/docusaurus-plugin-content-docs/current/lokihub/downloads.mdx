## نصب و استفاده

:::tip
اگر سیستم‌عامل شما فایل باینری را به‌عنوان تایید نشده (unverified) علامت‌گذاری کرد، ممکن است لازم باشد آن را صراحتاً در تنظیمات امنیتی خود مجاز کنید.
:::

### نسخه دسکتاپ (GUI)

**سیستم‌عامل مک (macOS)**
۱. فایل `.dmg` دانلود شده را باز کنید.
۲. برنامه **Lokihub** را به پوشه Applications خود بکشید.

**ویندوز**
۱. فایل `.zip` دانلود شده را از حالت فشرده خارج کنید.
۲. فایل `lokihub.exe` را اجرا کنید.

**لینوکس**
- **AppImage**:
  ```bash
  chmod +x lokihub-*.AppImage
  ./lokihub-*.AppImage
  ```
- **Debian/Ubuntu**:
  ```bash
  sudo dpkg -i lokihub-*.deb
  ```
- **Fedora/RHEL**:
  ```bash
  sudo rpm -i lokihub-*.rpm
  ```

### نسخه سرور (HTTP)

نسخه سرور به‌عنوان یک فرآیند پس‌زمینه اجرا می‌شود و رابط وب را از طریق پروتکل HTTP ارائه می‌دهد.

۱. **استخراج فایل آرشیو**:
   ```bash
   tar -xvf lokihub-server-*.tar.gz  # لینوکس
   # یا خارج کردن از حالت فشرده در ویندوز/مک
   ```

۲. **اجرای فایل باینری**:
   ```bash
   ./lokihub
   ```

۳. **باز کردن در مرورگر**:
   به آدرس `http://localhost:1610` (پورت پیش‌فرض) بروید.
