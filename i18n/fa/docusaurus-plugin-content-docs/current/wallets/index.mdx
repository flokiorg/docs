## مقایسه سریع

| ویژگی | tWallet | Komodo Wallet |
| :--- | :--- | :--- |
| **رابط** | ترمینال (TUI) | گرافیکی (GUI) |
| **Lightning** | بله (داخلی) | خیر |
| **تجربه** | متوسط | مبتدی |
| **بهترین برای** | اپراتورهای گره و حریم خصوصی | کاربران روزمره و موبایل |

:::tip[آیا از Lokihub استفاده می‌کنید؟]
اگر قصد استفاده از **Lokihub** را دارید، به شدت توصیه می‌کنیم با **tWallet v1** شروع کنید، زیرا backend Lightning لازم برای عملیات گره را فراهم می‌کند.
:::
