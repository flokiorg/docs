## مقایسه سریع

| ویژگی | Tap Wallet | tWallet | flnd | Komodo Wallet |
| :--- | :--- | :--- | :--- | :--- |
| **رابط** | وب / Mini App | ترمینال (TUI) | خط فرمان (CLI) | گرافیکی (GUI) |
| **Lightning** | خیر | بله (داخلی) | بله (بومی) | خیر |
| **تجربه** | مبتدی | متوسط | پیشرفته | مبتدی |
| **بهترین برای** | موبایل و اجتماعی | حریم خصوصی دسکتاپ | اپراتورهای گره | کاربران روزمره |

:::tip[از Lokihub استفاده می‌کنید؟]
اگر قصد استفاده از **Lokihub** را دارید، به شدت توصیه می‌کنیم با **tWallet v1** شروع کنید، زیرا backend Lightning لازم برای عملیات گره را فراهم می‌کند.
:::
