# تاریخچه نسخه‌های tWallet v0

این صفحه تاریخچه **tWallet v0 (Electrum)** را دنبال می‌کند. شاخه v0 برای کاربرانی نگهداری می‌شود که نیاز به **بازیابی بذر BIP39** دارند یا استفاده از سرورهای عمومی Electrum فلوکی‌کوین را ترجیح می‌دهند.

| نسخه | تمرکز |
| :--- | :--- |
| [**0.1.1-alpha**](/wallets/twallet/v0/0.1.1-alpha) | اضافه شدن قابلیت جابجایی بین سرورها در صورت بروز خطا و بهبود منطق همگام‌سازی کیف پول. |
| [**0.1.0-alpha**](/wallets/twallet/v0/0.1.0-alpha) | اولین نسخه آلفای عمومی برای رابط کاربری ترمینال با پشتیبانی از Electrum. |

---

:::important[به دنبال نسخه ۱ هستید؟]
اگر نیازی به بازیابی بذر BIP39 موجود ندارید، توصیه می‌کنیم برای داشتن بهترین عملکرد و حفظ حریم خصوصی، [**tWallet v1**](/wallets/twallet/v1) (Neutrino) را بررسی کنید.
:::
