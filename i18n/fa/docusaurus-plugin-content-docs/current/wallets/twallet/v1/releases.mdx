# تاریخچه نسخه‌های tWallet v1

| نسخه | تاریخ | تغییرات کلیدی |

| [**1.0.8-beta**](/wallets/twallet/v1/1.0.8-beta) | - | مهاجرت از Neutrino بنیاد btcsuite به نسخه اختصاصی Flokiorg جهت افزایش پایداری. |
