## تغییرات مهم

- به‌روزرسانی پارامترهای شبکه testnet3 و زنجیره رگرسیون (regression chain).
- اصلاح منطق سختی شبکه تست‌نت ۴ (testnet4).
- بازیابی محدودیت‌های پیش‌فرض PoW برای شبکه رگرسیون.
- به‌روزرسانی آستانه‌ها و زمان‌بندی‌های استقرار بیت‌های نسخه (version bits).
- حذف مکانیزم جایگزین (fallback) قدیمی برای محاسبه سختی.
- پاک‌سازی قالب‌بندی تست‌های نسخه بلوک.
