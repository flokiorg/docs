## 快速对比

| 功能 | tWallet | Komodo Wallet |
| :--- | :--- | :--- |
| **界面** | 终端（TUI） | 图形（GUI） |
| **Lightning** | 是（内置） | 否 |
| **使用门槛** | 中级 | 入门级 |
| **适用场景** | 节点运营者及注重隐私的用户 | 日常用户及移动端用户 |

:::tip[正在使用 Lokihub？]
如果您计划使用 **Lokihub**，强烈建议从 **tWallet v1** 开始，因为它为节点操作提供了必要的 Lightning 后端支持。
:::
