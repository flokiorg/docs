# tWallet v1 发布历史

本页面跟踪 **tWallet v1** 的演进历程。虽然为了安全性和性能，我们建议始终使用最新版本，但您可以在下方找到 v1 预发布版本的完整历史记录。

| 版本 | 日期 | 关键变更 |
| :--- | :--- | :--- |
| [**1.0.9-beta**](/wallets/twallet/v1/1.0.9-beta) | 最新 | 添加了 Windows 32 位资源并改进了依赖管理。 |
| [**1.0.8-beta**](/wallets/twallet/v1/1.0.8-beta) | - | 从 btcsuite 的 Neutrino 迁移到 Flokiorg 的分支版本，以增强稳定性。 |
| [**1.0.5-beta**](/wallets/twallet/v1/1.0.5-beta) | - | 修复了低内存设备上交易扫描的严重问题。 |
| [**1.0.4-beta**](/wallets/twallet/v1/1.0.4-beta) | - | 为移动端兼容的仪表板客户端优化了 gRPC 流。 |
| [**1.0.3-alpha**](/wallets/twallet/v1/1.0.3-alpha) | - | 初始 Flokicoin Neutrino 协议实现。 |
| [**1.0.2-alpha**](/wallets/twallet/v1/1.0.2-alpha) | - | 添加了 Segwit 和 Taproot 地址支持。 |
| [**1.0.1-alpha**](/wallets/twallet/v1/1.0.1-alpha) | - | 首次 WalletKit gRPC 服务公开测试。 |
