## 变更内容

### 依赖项
- 已更新 `go-flokicoin` → **[v0.25.7-beta](/lokichain/releases/0.25.7-beta)**
- 已更新 `flokicoin-neutrino` → **v0.16.2-beta**
- 已更新 `walletd` → **v0.1.3-beta**

### 修复
- 通用错误修复和稳定性改进。
- 提高了同步/初始化期间状态处理的可靠性。

## 说明
- 此版本与最新的上游库更新保持一致。
- 建议用户升级以获得更好的稳定性和兼容性。
