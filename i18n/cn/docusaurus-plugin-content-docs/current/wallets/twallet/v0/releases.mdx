# tWallet v0 发布历史

本页面跟踪 **tWallet v0 (Electrum)** 的历史记录。维护 v0 分支是为了满足需要进行 **BIP39 助记词恢复**或偏好使用公共 Flokicoin Electrum 服务器的用户。

| 版本 | 重点内容 |
| :--- | :--- |
| [**0.1.1-alpha**](/wallets/twallet/v0/0.1.1-alpha) | 添加了多服务器故障转移功能并改进了钱包同步逻辑。 |
| [**0.1.0-alpha**](/wallets/twallet/v0/0.1.0-alpha) | 具有 Electrum 支持的终端页面的首次公开 Alpha 测试。 |

:::important[正在寻找 v1？]
如果您不需要恢复现有的 BIP39 助记词，我们建议您查看 [**tWallet v1**](/wallets/twallet/v1)，以获得最佳的性能和隐私保护 (Neutrino)。
:::
