Lokinode 是一个跨平台桌面应用程序，为运行和管理 Flokicoin 全节点 (FLND) 提供图形界面。它允许您操作链上钱包并参与 Flokicoin 闪电网络 (FLN)。

## 功能特点

- **后台服务**：Lokinode 作为一个持久服务运行。您可以关闭用户界面，而节点将继续在系统托盘中运行。
- **链上钱包**：初始化并解锁您的钱包，发送/接收 FLC，并查看交易历史。
- **节点管理**：监控区块高度、同步状态和内存池提示。直接在界面中查看应用程序日志。
- **多节点配置文件**：在同一台机器上运行和管理多个节点。Lokinode 会自动跟踪您的数据目录和配置，允许您快速在不同的节点配置文件之间切换。

## 安装与设置

:::tip[从 tWallet 迁移？]
如果您已经在使用 [**tWallet**](/wallets/twallet)，**无需迁移**。您的 24 个助记词完全兼容。更棒的是，由于这两个应用程序共享相同的默认数据目录，Lokinode 将自动检测您现有的节点。您只需启动应用程序并开启默认节点配置文件，**无需任何设置**。
:::

1. **下载**：下载对应操作系统的可执行文件。
2. **节点配置**：启动 Lokinode。设置向导将首先要求您设置一个公共别名并选择本地数据目录。
3. **钱包设置**：选择创建新钱包（将提供新的 24 个助记词）或通过助记词恢复现有钱包。
4. **安全**：设置强加密密码以保护您的本地数据。
5. **同步**：节点启动后，将自动开始与 Flokicoin 区块链同步。

## 使用 Lokihub 进行管理

虽然 Lokinode 负责运行您的后台闪电节点并为基本链上钱包操作提供界面，但您必须使用 **Lokihub** 才能使用高级闪电网络功能。

当您在同一台机器上打开 Lokihub 时，它将自动尝试连接到您活跃的 Lokinode 目录，解锁以下功能：

- **流动性管理**：开启、关闭和监控闪电通道，以管理您的入站和出站容量。
- **子钱包**：创建专用钱包，为不同的应用程序或个人预算管理资金。
- **应用集成 (NWC)**：使用 Nostr Wallet Connect 将外部应用程序无缝连接到您的节点，并具有细粒度的权限控制。

[查看 Lokihub 设置指南 →](/lokihub/setup)

## 数据位置

默认情况下，Lokinode 将区块链和钱包数据存储在以下目录中：

| 操作系统 | 默认路径 |
| :--- | :--- |
| **Linux** | `~/.flnd/` |
| **macOS** | `~/Library/Application Support/Flnd/` |
| **Windows** | `%APPDATA%\Flnd\` |

除了这些默认路径外，您还可以在自定义位置创建新配置文件，或者通过将现有配置文件的数据目录复制到新路径，并通过应用程序的配置文件管理重新打开它来移动现有配置文件。

## 重要提示：闪电网络备份

您的 24 个助记词**仅用于恢复链上资金**。

要恢复锁定在已开启闪电通道中的资金，您需要**静态通道备份 (SCB)**。

:::danger[数据丢失风险]
请务必确保除了助记词之外，您还有一个单独的、经过验证的 `channel.backup` 文件备份。如果没有它，如果本地数据丢失，闪电通道中的资金将无法恢复。
:::
