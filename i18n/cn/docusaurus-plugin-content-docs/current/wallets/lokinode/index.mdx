## 安装

1.  **下载**：使用上面的按钮下载适合您操作系统的安装包。
2.  **执行安装程序**：
    *   **Windows**：运行 `.exe` 并按照设置向导操作。
    *   **macOS**：打开 `.dmg` 并将 Lokinode 拖到您的应用程序文件夹。
    *   **Linux**：通过您的软件包管理器安装 `.deb` 软件包。
3.  **设置**：启动应用程序以初始化您的节点。您可以创建一个新的 24 个单词的助记词或导入现有钱包。

## 能力

### Lightning Network (FLN) 运营
Lokinode 简化了高级 Lightning Network 管理：
*   **视觉渠道控制**：无需终端命令即可开启、关闭和监控渠道状态。
*   **流动性管理**：通过实时视觉反馈追踪入站和出站容量。
*   **结算**：生成并支付 Lightning 发票，并获得即时网络确认。

### 持久体验
专为节点可靠性而设计，Lokinode 将节点进程与图形界面分离：
*   **系统托盘集成**：从操作系统托盘控制节点状态并查看快速状态更新。
*   **非侵入式运行时**：关闭应用程序窗口会隐藏 UI，但保持节点在后台活动。
*   **自动稳定性**：内置逻辑处理网络中断并维持对等节点连接。

### 灵活配置
Lokinode 为需要控制其环境的高级用户而构建：
*   **自定义路径**：为区块链和渠道数据存储指定任何本地或外部目录。
*   **实时指标**：直接从仪表板监控区块高度、同步进度和对等节点数量。
*   **安全套件**：在安全的本地环境中管理加密钱包存储和恢复凭据。

## 使用 Lokihub 进行管理

虽然 **Lokinode** 为基础运营提供了内置界面，但高级运营商可以使用 **Lokihub** 进行详细的路由分析和远程节点管理。

### 快速连接
1.  **活动节点**：确保 Lokinode 正在运行且已同步。
2.  **连接**：打开 Lokihub；它将自动尝试从您的活动 Lokinode 目录检测凭据。
3.  **同步**：获得高级流动性图表和路由历史记录的访问权限。

## 数据位置

Lokinode 默认使用标准路径，但允许在初始设置期间或通过设置进行完全自定义。

| 操作系统 | 默认路径 |
| :--- | :--- |
| **Linux** | `~/.flnd/` |
| **macOS** | `~/Library/Application Support/Flnd/` |
| **Windows** | `%APPDATA%\Flnd\` |

## 极其重要：Lightning 备份

您的 24 个单词助记词**只能恢复链上资金**。 

1.  **静态渠道备份 (SCB)**：需要 `channel.backup` 文件来恢复开放的 Lightning 渠道中的资金。
2.  **安全要求**：您必须在助记词旁边保留 `channel.backup` 文件的独立备份。

:::danger[数据丢失风险]
始终确保您拥有经过验证的 `channel.backup` 文件备份。如果没有它，如果本地数据丢失，则无法恢复 Lightning 渠道资金。
:::
