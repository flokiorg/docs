Lokinode 是一个跨平台桌面应用程序，它为运行和管理 Flokicoin 全节点（FLND）提供了图形界面。它允许您操作链上钱包并参与 Flokicoin Lightning Network（FLN）。

## 功能

- **后台服务**：Lokinode 作为持久服务运行。您可以关闭用户界面，而节点将继续在系统托盘中运行。
- **链上钱包**：初始化和解锁您的钱包，发送/接收 FLC，并查看交易历史。
- **节点管理**：监控区块高度、同步状态和内存池顶端。直接在界面中查看应用程序日志。
- **多节点配置文件**：在同一台机器上运行和管理多个节点。Lokinode 会自动跟踪您的数据目录和配置，允许您在不同的节点配置文件之间快速切换。

## 安装与设置

:::tip[正在使用 tWallet 吗？]
如果您已经在使用 [**tWallet**](/wallets/twallet)，则**不需要任何迁移**。您的 24 个单词的助记词是完全兼容的。更好的是，由于这两个应用程序共享相同的默认数据目录，Lokinode 会自动检测到您现有的节点。您可以简单地启动该应用程序并启动您的默认节点配置文件，**完全无需设置**。
:::

1. **下载**：下载对应您操作系统的可执行文件。
2. **节点配置**：启动 Lokinode。设置向导将首先要求您设置一个公共别名并选择一个本地数据目录。
3. **钱包设置**：选择创建一个新钱包（它提供了一个新的 24 个单词的助记词），或通过助记词恢复现有钱包。
4. **安全性**：设置一个强大的加密密码以保护您的本地数据。
5. **同步**：节点启动后，它将自动开始与 Flokicoin 区块链同步。

:::info[面向节点运营者和开发者]
您是否希望在 VPS 上部署无头（headless）节点、运行自动化操作（Ops）或与 Docker 集成？您可能更愿意直接运行 [**`flnd`** 守护进程](/wallets/flnd)，而不是 Lokinode 图形界面。`flnd` 提供了核心后端，可以完全通过命令行进行管理。
:::

## 使用 Lokihub 进行管理

虽然 Lokinode 处理运行您的后台 Lightning 节点并提供基础链上钱包操作的界面，但您必须使用 **Lokihub** 来使用高级 Lightning Network 功能。

当您在同一台机器上打开 Lokihub 时，它会自动尝试连接到您活动的 Lokinode 目录，从而解锁以下功能：

- **流动性管理**：开启、关闭和监控 Lightning 渠道，以管理您的入站和出站容量。
- **子钱包**：创建专用的钱包来组织您的资金，用于不同的应用程序或个人预算。
- **应用集成（NWC）**：使用 Nostr Wallet Connect，通过细粒度的权限控制将外部应用程序无缝连接到您的节点。

[查看 Lokihub 设置指南 →](/lokihub/setup)

## 数据位置

默认情况下，Lokinode 将区块链和钱包数据存储在以下目录：

| 操作系统 | 默认路径 |
| :--- | :--- |
| **Linux** | `~/.flnd/` |
| **macOS** | `~/Library/Application Support/Flnd/` |
| **Windows** | `%APPDATA%\Flnd\` |

您可以在初始设置或应用程序设置中指定自定义数据路径。

## 极其重要：Lightning 备份

您的 24 个单词助记词**只能恢复链上资金**。 

要恢复锁定在开放 Lightning 渠道中的资金，您需要一个**静态渠道备份（SCB）**。 

:::danger[数据丢失风险]
始终确保您在助记词旁边有一个独立的、经过验证的 `channel.backup` 文件备份。如果没有它，如果本地数据丢失，则无法恢复 Lightning 渠道的资金。
:::
