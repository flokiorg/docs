## 快速设置指南

开始使用大约需要 5 分钟。请遵循以下简单步骤：

1. **获取应用**：在上方下载或打开适合您首选平台的版本。
2. **创建新钱包**：选择“Create New Wallet（创建新钱包）”并写下您的**助记词 (seed phrase)**。
3. **备份您的密钥**：将助记词记录在物理纸张上，并存放在安全的地方。
4. **启用 Flokicoin**：在资产列表中搜索 **Flokicoin** 并启用它以获取您的地址。

:::warning[安全第一]
助记词是您的*唯一*备份。如果您丢失了它，您将失去您的代币。切勿以数字方式存储、切勿共享，也切勿截图。
:::

## 关于地址 (F 与 fc1)

默认情况下，Komodo Wallet 生成以 **F** 开头的地址（Legacy/传统格式）。它也完全支持 **fc1** (Segwit) 地址。

* **Legacy (F)**：与旧版服务具有最大兼容性。
* **Segwit (fc1)**：推荐日常使用。它会产生**更低的网络费用**，且被大多数现代服务支持。

您可以在钱包设置中切换这些格式。两种格式在同一个 Flokicoin 网络上运行。
