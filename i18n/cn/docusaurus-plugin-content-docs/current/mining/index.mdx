## 开始赚取 Flokicoin

挖矿是您帮助保护网络并因此获得代币奖励的方式。Flokicoin 使用 **Scrypt** 算法，这意味着您可以在挖 Litecoin 和 Dogecoin 的同时挖 Flokicoin。

### 为什么选择挖 Flokicoin？

- **Scrypt 算法** — 加入 LTC 和 DOGE 共同使用的全球公认挖矿生态系统。
- **支持网络** — 每位矿工都有助于保持 Flokicoin 的去中心化和抗攻击性。
- **AuxPoW（辅助工作量证明，即合并挖矿）** — 在挖其他 Scrypt 币的同时"免费"赚取 Flokicoin。

:::warning[安全第一]
挖矿会大量占用硬件资源。请将矿机与个人数据隔离存放，确保设备散热良好，并只从可信来源下载软件。
:::

## 3 步快速入门

1. **选择硬件** — 认真挖矿选 Scrypt ASIC，爱好者挖矿用 GPU。
2. **加入矿池** — 稳定获得收益，无需等待数周才能独立挖出一个区块。[浏览矿池](/mining/pools)。
3. **指向即赚** — 将矿池的 stratum 地址和您的 FLC 钱包地址设为用户名即可开始。

## 网络基本参数

- **算法**：Scrypt
- **出块时间**：1 分钟
- **合并挖矿**：支持（在挖 LTC/DOGE 的同时免费赚取代币）

## 挖矿方式

### 加入矿池（推荐）

与他人合作共同找到区块。您将根据贡献比例获得奖励分成。[查看矿池列表](/mining/pools)。

### 单独挖矿（进阶）

独自挖矿。您可独享每个区块的全部奖励，但出块间隔可能较长。

### 合并挖矿（AuxPoW）

如果您的矿池支持，可以在挖 Litecoin 或 Dogecoin 的*同时*挖 Flokicoin，无需消耗任何额外电力。

## 挖矿软件
