# 功能特性

Lokihub 提供了运行自托管闪电网络节点所需的一切。

## 闪电网络钱包

通过闪电网络通道和链上交易发送及接收资金。

**功能：**
- **发送与接收** — 创建闪电网络发票并向他人支付
- **链上交易** — 进行常规的 Flokicoin 交易
- **子钱包** — 为不同用途创建独立的钱包
- **历史记录** — 查看所有链上和闪电网络交易

:::tip
Lokihub 的默认货币单位是 **Loki**。`1 FLC = 100,000,000 Loki`。
:::

## 通道管理

闪电网络通道让您可以进行即时支付。Lokihub 帮助您管理这些通道。

**功能：**
- **开启通道** — 连接到其他节点以启用路由
- **关闭通道** — 关闭通道并将资金返回链上
- **手动控制** — 向特定对等节点开启/关闭通道
- **流动性可视化** — 查看您可以发送（出站）与接收（入站）的金额


## 对等节点管理

查看并管理您与闪电网络的连接。

- 查看已连接的对等节点
- 手动添加新的对等节点
- 断开不再需要的对等节点连接
- 更新节点别名

## 应用连接 (NWC)

使用 [**Nostr WalletConnect**](https://nwc.dev) (NWC) 将外部应用连接到您的闪电网络节点。

NWC 是一个开放标准，允许应用在不持有您私钥的情况下请求从您的钱包支付。您可以授予权限，并随时撤销访问权限。

**工作原理：**
1. 打开支持 NWC 的应用
2. 在 Lokihub 中生成连接字符串
3. 扫描二维码或在应用中粘贴连接字符串
4. 授予权限（只读、发送最多 X 金额等）
5. 使用该应用——它现在可以请求从您的节点支付

**连接示例：**
- 需要打赏/支付小费的社交应用
- 需要支付授权的服务
- 任何兼容 NWC 的应用程序

在 [nwc.dev](https://nwc.dev) 了解更多关于 NWC 标准的信息。

## 服务自由

Lokihub 的设计初衷是社区驱动。用户可以提交 PR 来添加他们自己的服务。选择权在您手中。

**可配置服务：**
- **LSP** — 为入站流动性提供支持的闪电网络服务商 (Lightning Service Provider)
- **Nostr Relay** — 为 NWC 连接提供消息路由
- **区块浏览器** — 交易查询与监控

您可以随时在“服务”设置中切换服务商。[了解更多 →](/lokihub/services)
