# 掌控您的闪电节点

**Lokihub** 让您对闪电节点拥有完整的金融主权。它不仅仅是一个管理器，更是您通往去中心化网络的门户——与 [tWallet](/wallets/twallet) 深度集成，帮助您管理子钱包、处理通道，并通过 NWC（Nostr 钱包连接协议）连接应用程序。

:::caution[重要提示：备份风险]
**闪电通道资金不存储在区块链上。** 24 个助记词仅能恢复您的链上余额。为避免通道资金彻底损失，您必须备份**静态通道备份（SCB）**。
[了解如何保护您的节点 →](/wallets/twallet/v1#critical-lightning-backups)
:::
