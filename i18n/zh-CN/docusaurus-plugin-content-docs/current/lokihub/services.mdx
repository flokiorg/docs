# Lokihub 服务

Lokihub 使用外部服务进行闪电网络操作、[NWC](https://nwc.dev) 通信和区块链查询。您可以选择社区默认配置，也可以自行配置。

## 服务类型

### LSP (闪电网络服务提供商)

**可选配置** — 提供用于接收支付的入站流动性。

当您开启一个闪电网络通道时，您只能发送自己存入的资金。要接收支付，您需要入站流动性——这可以来自自然的通道平衡，也可以来自 LSP 为您开启的通道。

**LSP 提供以下服务：**
- 为您的节点开启通道
- 提供入站流动性
- 即时 (JIT) 通道创建

**在 Lokihub 中：**
从社区 LSP 中选择，或添加您自己的 LSP。Lokihub LSP 使用 **LSPS 规范** ([LSPS0 到 LSPS5](https://github.com/lightning/blips)) 以实现标准化的流动性。

### Nostr 中继器

在您的节点和已连接的应用之间路由 [NWC](https://nwc.dev) 连接消息。

当您通过 NWC 连接应用时，您的节点和应用通过 Nostr 中继器进行通信。中继器无法看到支付数据——它只处理加密的连接消息。

**在 Lokihub 中：**
选择一个公共中继器或运行您自己的中继器。您可以选择多个中继器，以确保强大的通信备用方案和可靠性。

### 区块浏览器

查询链上交易详情、区块链信息和闪电网络节点统计数据。

**在 Lokihub 中：**
指向任何兼容 Flokicoin 的区块浏览器。

### 小组件留言板

交互式的聊天小组件，社区成员可以通过打赏 (zaps) 发布消息。收集到的资金可用于社区空投或其他用途。您可以为不同的频道配置多个小组件。

**在 Lokihub 中：**
从社区留言板中选择，或配置您自己的留言板。

### 兑换服务 (Swap Service)

**可选配置** — 在您的链上钱包和闪电网络通道之间兑换资金。

当您需要将资金从链上余额移动到闪电网络 (loop in)，或从闪电网络移动到链上 (loop out) 时非常有用。

**在 Lokihub 中：**
使用由您选择的兑换提供商支持的内置兑换界面。

## 服务商：提交您的服务

服务目录是开源的，并由社区维护。

### 仓库结构

服务配置位于：[flokiorg/lokihub-services](https://github.com/flokiorg/lokihub-services)

```
lokihub-services/
├── services.json      # LSP、中继器、浏览器
├── rates.json         # 汇率
├── currencies.json    # 支持的货币
└── info.json          # 通用信息和 MOTD
```

### 添加服务

**1. Fork 仓库**

在 GitHub 上 Fork [flokiorg/lokihub-services](https://github.com/flokiorg/lokihub-services)。

**2. 编辑 `services.json`**

将您的服务添加到相应的板块。

#### 添加 LSP

添加到 `lsp` 数组：

```json
{
  "name": "My LSP",
  "uri": "pubkey@host:port",
  "description": "支持 JIT 通道和入站流动性"
}
```

**字段说明：**
- `name` — LSP 的显示名称
- `uri` — 连接字符串 (`pubkey@host:port`)
- `description` — 所提供服务的简要描述

#### 添加 Nostr 中继器

添加到 `relay` 数组：

```json
{
  "name": "My Relay",
  "url": "wss://relay.example.com",
  "description": "适用于 NWC 的快速 Nostr 中继器"
}
```

**字段说明：**
- `name` — 显示名称
- `url` — WebSocket URL
- `description` — 简要描述

#### 添加区块浏览器

添加到 `explorer` 数组：

```json
{
  "name": "My Explorer",
  "url": "https://explorer.example.com",
  "description": "Flokicoin 区块链浏览器"
}
```

**字段说明：**
- `name` — 显示名称
- `url` — HTTPS URL
- `description` — 简要描述

#### 添加小组件留言板

添加到 `widget` 数组：

```json
{
  "name": "Community Board",
  "url": "nostr+walletconnect://b889...4f1?relay=wss://relay.example.com/v1&secret=79...a1",
  "description": "该留言板收集的资金将用于空投活动和游戏奖励"
}
```

**字段说明：**
- `name` — 显示名称
- `url` — NWC 连接字符串 (`nostr+walletconnect://...`)
- `description` — 简要描述

**3. 创建 Pull Request**

提交更改以供审核。

### 审核流程

Pull request 将针对以下内容进行审核：
- 有效的 JSON 结构
- 可用的服务终端
- 准确的服务信息
- 可靠性和在线时间（针对社区默认项）

## 文件说明

### `services.json`

推荐的社区服务目录：
- **LSPs** — 闪电网络服务提供商
- **Relays** — 用于 NWC 的 Nostr 中继器
- **Explorers** — 区块浏览器
- **Widgets** — 带有打赏功能的聊天留言板
- **Swap Services** — 链上/链下资金兑换

### `rates.json`

FLC (Flokicoin) 对法定货币的最新汇率。

### `currencies.json`

支持的法定货币列表及其符号。

### `info.json`

面向 Lokihub 用户的通用信息和每日消息 (MOTD)。
