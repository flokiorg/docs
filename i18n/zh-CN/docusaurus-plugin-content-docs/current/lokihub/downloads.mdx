## 安装与使用

:::tip
如果您的操作系统将该二进制文件标记为未验证，您可能需要在安全设置中手动允许其运行。
:::

### 桌面版 (GUI)

**macOS**
1. 打开下载的 `.dmg` 文件。
2. 将 **Lokihub** 拖动到您的“应用程序 (Applications)”文件夹中。

**Windows**
1. 解压下载的 `.zip` 文件。
2. 运行 `lokihub.exe`。

**Linux**
- **AppImage**:
  ```bash
  chmod +x lokihub-*.AppImage
  ./lokihub-*.AppImage
  ```
- **Debian/Ubuntu**:
  ```bash
  sudo dpkg -i lokihub-*.deb
  ```
- **Fedora/RHEL**:
  ```bash
  sudo rpm -i lokihub-*.rpm
  ```

### 服务端 (HTTP)

服务端版本作为后台进程运行，并通过 HTTP 提供 Web 界面。

1. **解压归档文件**：
   ```bash
   tar -xvf lokihub-server-*.tar.gz  # Linux
   # 或者在 Windows/macOS 上解压
   ```

2. **运行二进制文件**：
   ```bash
   ./lokihub
   ```

3. **在浏览器中打开**：
   访问 `http://localhost:1610`（默认端口）。
