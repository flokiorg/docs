这是一个用于测试和反馈的**预发布版本**。鼓励开发者和早期采用者**报告问题**。

## 变更日志

- 将后端从 **Electrum** 迁移到 **Neutrino**。
- 集成了具有完整闪电网络技术栈支持的 **Flokicoin 闪电网络节点 (LN)**。
- 实现了在 Flokicoin 网络上的**原生闪电网络开发**。
- 添加了可配置的地址类型：`segwit`、`nested-segwit`、`taproot`。
- 引入了安全的钱包锁定/解锁以及密码修改 UI。
- 改进了具有即时费率估计功能的交易构建器。
- 增强了**余额显示**，包含已确认和未确认金额。
- 增强了具有同步状态、闪电网络健康状况和实时更新功能的终端 UI。
- 进行了重大的内部重构，用于模块化配置、增强稳定性以及未来扩展。
