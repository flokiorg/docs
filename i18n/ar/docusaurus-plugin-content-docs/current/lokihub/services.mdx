# خدمات Lokihub

يستخدم Lokihub خدمات خارجية لعمليات Lightning، واتصالات [NWC](https://nwc.dev)، والاستعلام عن البلوكشين. يمكنك الاختيار من بين الإعدادات الافتراضية للمجتمع أو تكوين إعداداتك الخاصة.

## أنواع الخدمات

### LSP (مزود خدمة Lightning)

**اختياري** — يوفر السيولة الواردة لاستلام المدفوعات.

عندما تفتح قناة Lightning، يمكنك فقط إرسال الأموال التي أودعتها. لاستلام المدفوعات، تحتاج إلى سيولة واردة — إما من الموازنة الطبيعية للقناة أو من خلال قيام LSP بفتح قناة إليك.

**ما يقدمه مزودو LSP:**
- فتح قنوات لعقدتك
- توفير السيولة الواردة
- إنشاء القنوات عند الطلب (JIT)

**في Lokihub:**
اختر من بين مزودي LSP في المجتمع أو أضف مزودك الخاص. يستخدم مزودو LSP في Lokihub **مواصفات LSPS** (من [LSPS0 إلى LSPS5](https://github.com/lightning/blips)) لسيولة موحدة.

### مرحل Nostr (Nostr Relay)

يوجه رسائل اتصال [NWC](https://nwc.dev) بين عقدتك والتطبيقات المتصلة.

عندما تربط تطبيقًا عبر NWC، تتواصل عقدتك والتطبيق عبر مرحل Nostr. لا يرى المرحل بيانات الدفع — فقط رسائل الاتصال المشفرة.

**في Lokihub:**
اختر مرحلاً عامًا أو قم بتشغيل مرحلك الخاص. يمكنك تحديد أكبر عدد تريده من المرحلات لضمان بدائل اتصال قوية وموثوقية عالية.

### مستكشف الكتل (Block Explorer)

للبحث عن تفاصيل المعاملات على الشبكة (on-chain)، ومعلومات البلوكشين، وإحصائيات عقدة Lightning.

**في Lokihub:**
قم بالإشارة إلى أي مستكشف كتل متوافق مع Flokicoin.

### لوحة رسائل الويدجت (Widget Messageboard)

ويدجت دردشة تفاعلية حيث يمكن لأعضاء المجتمع نشر رسائل مع "zaps" (إكراميات). يمكن استخدام الأموال المجمعة للإنزال الجوي (airdrop) للمجتمع أو لأغراض أخرى. قم بتكوين ويدجت متعددة لقنوات مختلفة.

**في Lokihub:**
اختر من بين لوحات رسائل المجتمع أو قم بتكوين لوحتك الخاصة.

### خدمة التبديل (Swap Service)

**اختياري** — تبديل الأموال بين محفظتك على الشبكة (on-chain) وقنوات Lightning.

مفيد عندما تحتاج إلى نقل الأموال من رصيدك على الشبكة إلى Lightning (loop in) أو من Lightning إلى الشبكة (loop out).

**في Lokihub:**
استخدم واجهة التبديل المدمجة والمدعومة من مزود التبديل الذي اخترته.

## لمزودي الخدمة: أرسل خدمتك

دليل الخدمات مفتوح المصدر ويديره المجتمع.

### هيكل المستودع

توجد تكوينات الخدمة في: [flokiorg/lokihub-services](https://github.com/flokiorg/lokihub-services)

```
lokihub-services/
├── services.json      # مزودو LSP، المرحلات، المستكشفون
├── rates.json         # أسعار الصرف
├── currencies.json    # العملات المدعومة
└── info.json          # معلومات عامة ورسالة اليوم (MOTD)
```

### إضافة خدمة

**1. عمل Fork للمستودع**

قم بعمل Fork لمستودع [flokiorg/lokihub-services](https://github.com/flokiorg/lokihub-services) على GitHub.

**2. تحرير ملف `services.json`**

أضف خدمتك إلى القسم المناسب.

#### إضافة LSP

أضف إلى مصفوفة `lsp`:

```json
{
  "name": "My LSP",
  "uri": "pubkey@host:port",
  "description": "يدعم قنوات JIT والسيولة الواردة"
}
```

**الحقول:**
- `name` — الاسم المعروض لمزود LSP الخاص بك
- `uri` — سلسلة الاتصال (`pubkey@host:port`)
- `description` — وصف موجز للخدمات المقدمة

#### إضافة مرحل Nostr

أضف إلى مصفوفة `relay`:

```json
{
  "name": "My Relay",
  "url": "wss://relay.example.com",
  "description": "مرحل Nostr سريع لـ NWC"
}
```

**الحقول:**
- `name` — الاسم المعروض
- `url` — رابط WebSocket
- `description` — وصف موجز

#### إضافة مستكشف كتل

أضف إلى مصفوفة `explorer`:

```json
{
  "name": "My Explorer",
  "url": "https://explorer.example.com",
  "description": "مستكشف بلوكشين Flokicoin"
}
```

**الحقول:**
- `name` — الاسم المعروض
- `url` — رابط HTTPS
- `description` — وصف موجز

#### إضافة لوحة رسائل ويدجت

أضف إلى مصفوفة `widget`:

```json
{
  "name": "Community Board",
  "url": "nostr+walletconnect://b889...4f1?relay=wss://relay.example.com/v1&secret=79...a1",
  "description": "الأموال المجمعة في لوحة الرسائل هذه مخصصة لجلسات الإنزال الجوي (airdrop) ومدفوعات الألعاب"
}
```

**الحقول:**
- `name` — الاسم المعروض
- `url` — سلسلة اتصال NWC (`nostr+walletconnect://...`)
- `description` — وصف موجز

**3. إنشاء طلب سحب (Pull Request)**

أرسل تغييراتك للمراجعة.

### عملية المراجعة

تتم مراجعة طلبات السحب من حيث:
- صحة هيكل JSON
- عمل نقاط نهاية الخدمة بشكل صحيح
- دقة معلومات الخدمة
- الموثوقية ووقت التشغيل (للإعدادات الافتراضية للمجتمع)

## وصف الملفات

### `services.json`

دليل الخدمات المجتمعية الموصى بها:
- **LSPs** — مزودو خدمة Lightning
- **Relays** — مرحلات Nostr لـ NWC
- **Explorers** — مستكشفو الكتل
- **Widgets** — لوحات رسائل دردشة مع zaps
- **Swap Services** — تبديل الأموال بين الشبكة وخارجها

### `rates.json`

أسعار الصرف الحالية لـ FLC (Flokicoin) مقابل العملات الورقية.

### `currencies.json`

قائمة العملات الورقية المدعومة ورموزها.

### `info.json`

معلومات عامة ورسالة اليوم (MOTD) لمستخدمي Lokihub.
