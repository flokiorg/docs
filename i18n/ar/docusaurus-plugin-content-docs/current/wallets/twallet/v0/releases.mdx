# سجل إصدارات tWallet v0

تتتبع هذه الصفحة سجل إصدارات **tWallet v0 (Electrum)**. يتم صيانة الفرع v0 للمستخدمين الذين يحتاجون إلى **استعادة بذرة BIP39** أو يفضلون استخدام خوادم Electrum العامة لـ Flokicoin.

| الإصدار | التركيز |
| :--- | :--- |
| [**0.1.1-alpha**](/wallets/twallet/v0/0.1.1-alpha) | إضافة ميزة تجاوز فشل الخوادم المتعددة وتحسين منطق مزامنة المحفظة. |
| [**0.1.0-alpha**](/wallets/twallet/v0/0.1.0-alpha) | أول نسخة ألفا عامة لواجهة الطرفية (Terminal) مع دعم Electrum. |

---

:::important[هل تبحث عن v1؟]
إذا لم تكن بحاجة إلى استعادة بذرة BIP39 موجودة، فنحن نوصي بالاطلاع على [**tWallet v1**](/wallets/twallet/v1) للحصول على أفضل أداء وخصوصية (Neutrino).
:::
