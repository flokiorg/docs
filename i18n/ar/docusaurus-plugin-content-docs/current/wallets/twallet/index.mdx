## البدء

بمجرد اختيار الإصدار، يستغرق الإعداد حوالي 5 دقائق:

1. **حمّل** الإصدار الخاص بنظام تشغيلك.
2. **انسخ احتياطياً** عبارة الاسترداد الخاصة بك. اكتبها و **لا تشاركها أبداً**.
3. **التزامن** — اترك المحفظة لتلحق بالشبكة (بضع دقائق كحد أقصى).
4. **الاستلام** — قم بتوليد عنوان واحصل على عملاتك الأولى.
