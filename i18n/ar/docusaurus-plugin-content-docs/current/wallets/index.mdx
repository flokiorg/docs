## مقارنة سريعة

| الميزة | tWallet | Komodo Wallet |
| :--- | :--- | :--- |
| **الواجهة** | طرفية (TUI) | رسومية (GUI) |
| **Lightning** | نعم (مدمجة) | لا |
| **مستوى الخبرة** | متوسط | مبتدئ |
| **الأفضل لـ** | مشغّلو العقد والخصوصية | المستخدمون اليوميون والجوّال |

:::tip[هل تخطط لاستخدام Lokihub؟]
إذا كنت تخطط لاستخدام **Lokihub**، نوصي بشدة بالبدء بـ **tWallet v1**، إذ يوفر الواجهة الخلفية اللازمة لـ Lightning لعمليات العقدة.
:::
